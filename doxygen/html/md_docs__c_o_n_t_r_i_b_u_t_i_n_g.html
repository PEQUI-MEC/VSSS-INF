Obrigado por considerar contribuir com os projetos do pequi mecânico.

1. Para onde ir?

Caso tenha encontrado algum bug ou tenha alguma sujestão, entre em issues, e verifique se algo parecido já foi reportado, em caso positivo, contribua com um comentário, caso contrário, crie um novo issue ou entre em contato com pqver.nosp@m.ysma.nosp@m.ll@gm.nosp@m.ail..nosp@m.com.

2. Contribuições de código

Caso tenha encontrado um issue aceito que queira contribuir, comente no mesmo avisando do interesse, caso aprovado, faça um fork e crie um branch com um nome descritivo para a solução, por exemplo o issue 65:

git checkout -b 65-traduzir-para-portugues

Quando terminado, faça um pull request. Baseado em:

  • https://github.com/activeadmin/activeadmin/blob/master/CONTRIBUTING.md
  • https://guides.github.com/introduction/flow/
